# zsh

## Installation

- download `.zsh-extend` files
- copy to `~/.zsh/.zsh-extend`
- download `.zsh-aliases` files
- copy to `~/.zsh/.zsh-aliases`
- add following code to `~/.zshrc`

```
############################################
# LOAD CUSTOM CONFIG
############################################
source ~/.zsh/.zsh-extend
source ~/.zsh/.zsh-aliases
```
